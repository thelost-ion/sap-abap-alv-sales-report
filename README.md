# SAP ABAP ALV Report - Sales Order Analysis

## Overview
An ABAP report that fetches and displays sales orders from
VBAK and VBAP using ALV Grid View.

## Features
- Selection screen filtering (Sold-to, Material, Date range)
- INNER JOIN for VBAK + VBAP
- ALV Grid with sorting and filtering

## How to Run
1. Upload ZSD SALES ALV REPORT abap via SE38
2. Activate and Execute (F8)
3. Enter filter criteria on the selection screen
4. View results in ALV grid

Sales | Order | Header
Sales | Order | Items
